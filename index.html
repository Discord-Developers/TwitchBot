Commands
Here's a short introduction of the most relevant commands (All commands can be seen with 't!help'!)

Administrative Commands

  • create-channel: Create a text channel.
  • purge [1-100]: Delete a specified number of messages, including pinned messages.
  • broadcast [message]: Send a global broadcast.
  • tempban [user] [duration]: Temporarily ban a user from the guild.
  • ban [user]: Ban a user from the guild.
  • unban [user]: Unban a user from the guild.

Moderation Commands

  • announce [channel] [message]: Send an announcement to a specified channel.
  • warn [user]: Warn a user.
  • mute [user]: Mute a user.
  • tempmute [user] [duration]: Temporarily mute a user.
  • prune [1-100]: Deletes a specified amount of messages.

Utility Commands

  • help: Displays a list of available commands, or detailed information for a specified command.
  • prefix: Shows or sets the command prefix.
  • ping: Checks the bot's ping to the Discord server.
  • user-info: Gets information about a user.

Twitch Commands

  • stream [add|remove|list] [user] [twitch-channel]: Add or Remove a user to the list of guild stream alerts.
  • chat [add|remove|list] [channel] [twitch-channel]: Add or remove a Twitch Chat Channel.
